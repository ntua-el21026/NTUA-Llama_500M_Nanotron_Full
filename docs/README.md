# Docs Folder

This folder contains report material and repository maintenance utilities.

## Paper

- `paper/main.tex`: LaTeX source of the report
- `paper/Report.pdf`: compiled report

## Maintenance utilities

- `maintain.py`: runs local documentation maintenance tasks
- `project_structure/project_structure.py`: generates `project_structure.txt`
- `project_analytics/project_analytics.py`: generates `project_analytics.txt`

The maintenance scripts are configured for this repository layout.
